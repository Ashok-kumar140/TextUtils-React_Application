# TextUtils - Text Utility Web Application using React..



Click The Link to Check My Webstite [click here](https://mytextutility1.netlify.app/).

## Available Features

In this application following features added:

1) Convert text into UpperCase.
2) Convert text into LowerCase.
3) Remove extra spaces.
4) Copy the text.
5) Save the text file.
6) Clear the text
7) Count the word and characters
8) Average time required to read the text

## Requirement

1) HTML
2) CSS
3) Java-script
4) React

## Steps to run this application in your PC
Open terminal and run following commands
1) npm install
2) npm run start
